# Panduan Berkontribusi

Terima kasih telah tertarik untuk berkontribusi!

## Melaporkan Bug
Gunakan tab **Issues** untuk melaporkan bug. Sertakan:
1. Deskripsi yang jelas.
2. Langkah-langkah untuk mereproduksi bug.

## Mengusulkan Fitur
Gunakan tab **Issues** dengan label `enhancement`. Jelaskan ide Anda.

## Membuat Perubahan
1. **Fork** repositori ini.
2. Buat **Branch** baru: `git checkout -b fitur/nama-fitur`.
3. Lakukan **Commit** perubahan Anda.
4. Buka **Pull Request (PR)** ke branch `main` di repositori ini.

Kami menghargai kontribusi Anda!
